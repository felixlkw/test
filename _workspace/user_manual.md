# SafeMate 사용 설명서

오늘 처음 SafeMate를 켜는 분을 위한 단계별 안내서입니다. 화면 흐름대로 따라가면 됩니다. 음성으로 답할 때는 책 표시(💬)된 예시 멘트를 그대로 따라 말해도 좋고, 본인 현장 상황에 맞춰 바꿔 말해도 됩니다.

> **앱은 4가지 산업을 지원합니다** — 제조 / 건설 / 중공업 / 반도체. 어느 산업이든 화면 흐름은 똑같습니다. 다른 점은 **선택할 작업 종류**와 **AI가 묻는 질문**뿐입니다.

---

## 시작하기 전에

1. 백엔드가 가동 중인지 확인 (PC라면 터미널에 `uvicorn` 실행 중).
2. 브라우저에서 `http://localhost:5173` 열기.
3. 처음이라면 브라우저가 **마이크/카메라 사용 권한**을 묻습니다 → **허용** 누르세요.
4. 화면 상단에 PwC 로고와 **SafeMate** 글씨가 보이면 준비 완료입니다.

---

## 1단계 — 새 TBM 시작

화면 가운데 큰 주황색 **"새 TBM 시작"** 버튼을 누릅니다.

아래에서 **산업 도메인 선택** 시트가 올라옵니다. 본인 현장에 맞는 도메인 카드를 누르세요.

| 도메인 | 누르는 카드 | 어떤 분이 사용하나요 |
|---|---|---|
| 제조 | 🏭 제조 | 가전·금속·조립·포장 라인의 안전관리자 |
| 건설 | 🏗️ 건설 | 신축·리모델링·토목·플랜트 현장 |
| 중공업 | ⚓ 중공업 | 조선·해양·철강·중장비 야드 |
| 반도체 | 🔬 반도체 | FAB·후공정·가스/화학 클린룸 |

> 💡 자주 한 도메인만 사용하신다면 **설정 → 운영 → 기본 도메인**에 등록해 두세요. 다음부터는 시트가 안 뜨고 바로 다음 화면으로 넘어갑니다.

---

## 2단계 — 오늘 할 작업 고르기

"준비" 화면이 뜨면 **작업 선택** 섹션에서 카드를 하나 누릅니다. 각 도메인마다 3개의 작업 종류가 준비되어 있습니다.

| 도메인 | 작업 카드 (예시 — 본인 현장에 맞는 것 선택) |
|---|---|
| 제조 | 조립라인 작업 / 지게차 운반작업 / **프레스 작업** |
| 건설 | **고소작업** / 화기작업 / 굴착작업 |
| 중공업 | 양중작업 / 화기작업 / **밀폐공간 작업** |
| 반도체 | **CVD 챔버 PM** / 가스 라인 브레이크 / 리소 장비 PM |

**굵은 글씨**가 이 매뉴얼의 예시로 사용할 작업입니다. 본인 작업이 다르면 다른 카드를 골라도 흐름은 동일합니다.

카드를 누르면 잠시 **"위험 추천 불러오는 중…"** 메시지가 뜨고, 1~3초 후에 AI가 위험 항목을 자동으로 채워 줍니다.

---

## 3단계 — 현장 상황 알려주기 (선택, 권장)

작업 선택 아래 펼쳐진 **현장 상황** 폼에 오늘 현장 정보를 적습니다. 더 정확한 위험 추천을 받기 위함입니다.

| 입력 칸 | 어떻게 채우나요 | 예시 |
|---|---|---|
| 작업자 수 | 오늘 작업할 사람 수 | 4 |
| 교대 | 주간 / 야간 / 특근 등 | 주간 |
| 풍속 (m/s) | 옥외 작업이면 입력, 옥내면 비워둠 | 12 (건설 옥외) |
| 신규 자재 | 평소와 다른 자재 들어오면 적기 | 알루미늄 5052 판재 |
| 특이사항 | 평소와 달라진 점, 우려사항 | 금형 교체 1회 예정 |
| 과거 사고 키워드 | 최근 1개월 내 사고 키워드 | 협착, 면장갑 말림 |

> ⚠️ **반도체 현장은 이 폼이 회색으로 비활성화되어 있습니다.** 영업비밀 보호 기본값입니다. 입력 없이도 작동하니 그대로 다음 단계로 넘어가세요. 입력하고 싶다면 **설정 → AI 컨텍스트 활용 → 반도체 ON** 후에 가능합니다.

입력 후 폼 우측 상단의 **🔄 다시 받기** 버튼을 한 번 눌러야 입력 내용이 추천에 반영됩니다(자동 반영 안 됨 — API 보호 차원).

---

## 4단계 — AI 위험 추천 검토

화면 아래로 스크롤하면 **위험 추천** 섹션이 보입니다.

- **자물쇠 🔒 표시 항목**: 필수 점검입니다. 끄거나 건너뛸 수 없습니다.
- **회색 톤 항목**: 컨텍스트(풍속·특이사항 등)에 따라 추가된 조건부 항목입니다.
- 각 항목 아래 작은 글씨: 산업안전보건기준규칙 §XX 같은 **법규 출처** 와 catalog/llm 칩.

가볍게 훑어보고, 그 아래 **추천 질문** 4~6개도 둘러봅니다. 음성 대화에서 작업자에게 묻기 좋은 예시 질문들입니다.

준비됐으면 화면 맨 아래 주황색 **"TBM 시작"** 버튼을 누릅니다.

---

## 5단계 — 음성 대화 시작

음성 화면(`/run`)으로 자동 전환됩니다. 잠깐 기다리면 AI가 알아서 연결됩니다(보통 3~5초).

**처음 보이는 것**
- 상단: 모드 칩(TBM/EHS 토글) / 언어 칩(한국어) / 우측 **TBM은 "정리본 + 종료" / EHS는 "종료"만**
- 가운데(TBM만): 진행 단계 4칸 — **사전 정보 → 점검 → 대응 → 정리**
- AI 첫 인사말이 채팅창에 뜸
- 하단: 좌측 마이크 버튼 / 가운데 텍스트 입력 / 우측 카메라 버튼

> 💡 **모드 칩으로 전환할 때** — 상단의 TBM/EHS 토글을 클릭하면 새 세션을 시작하는 것과 동일한 흐름으로 진입합니다.
> - **TBM 클릭** (현재 EHS): 기본 도메인이 설정돼 있으면 새 TBM 세션 + "준비" 화면으로 직행, 없으면 홈으로 이동 + **산업군 선택 시트 자동 노출**.
> - **EHS 클릭** (현재 TBM): 새 EHS 세션 생성 + EHS 챗 화면 진입.
>
> 즉 모드 칩은 "현재 화면에서 모드만 토글"이 아니라 "다른 모드의 새 세션 시작 흐름"입니다 — 진행 중 세션은 먼저 보존됩니다(과거 기록에서 확인 가능).

화면에 **"🎙️ 마이크 켜기"** 안내 배너가 뜨면, 좌측 하단 **마이크 버튼** 을 누릅니다. 버튼이 주황색으로 변하면 음성 대화 가능 상태입니다.

> 💡 음성이 부담스러우면 가운데 텍스트 입력칸에 글로 적어도 됩니다. 둘 다 똑같이 작동합니다.

## 6단계 — 사전 정보 답하기 (음성)

AI가 "오늘 어떤 작업을 진행하시나요?" 같은 질문으로 시작합니다. 아래 예시 멘트 중 본인 도메인에 맞는 것을 따라 말해 보세요.

**도메인별 첫 멘트 예시**

🏭 **제조 (프레스 작업)**
> 💬 *"오늘 프레스 작업이고요, 작업자 4명입니다. 신규 자재로 알루미늄 5052 판재 들어옵니다. 금형 교체도 한 번 있어요. 지난주에 옆 라인에서 협착 사고가 있어서 신경 쓰입니다."*

🏗️ **건설 (고소작업)**
> 💬 *"오늘은 5층 외벽 마감 작업입니다. 작업자 6명이고, 풍속이 12 m/s 정도로 강해요. 어제 옆 현장에서 공구 낙하 사고가 있었습니다."*

⚓ **중공업 (밀폐공간 작업)**
> 💬 *"오늘 블록 내부 도장 점검 작업입니다. 작업자 3명, 블록 탱크 안에 진입할 거예요. 유기용제 도료 사용하고요. 인접 야드에서 1주 전 산소 결핍 사고가 있었어요."*

🔬 **반도체 (CVD 챔버 PM)**
> 💬 *"오늘 야간 CVD 챔버 PM입니다. 작업자 2명이고, SiH4·NH3 라인 다룹니다. 한 달 전 인접 챔버에서 PFC 노출이 있어서 좀 신경 쓰입니다."*

AI가 한 번에 한 가지씩 추가 질문을 합니다 (작업 시간대, 위험 인지도 등). 짧게 사실대로만 답하면 됩니다.

> 💡 AI가 모르는 단어를 말하면 다시 물어볼 수 있어요. "잘 못 들었는데, SiH4가 뭐예요?" 처럼요.

## 7단계 — 체크리스트 답하기 (음성)

AI가 5개 안전 점검 항목을 하나씩 묻습니다. **현장 상태를 짧게 답하면** AI가 자동으로 체크해 줍니다. 화면 상단 게이지가 **0/5 → 5/5** 로 올라갑니다.

**도메인별 체크리스트 응답 예시**

🏭 **제조 (프레스 작업)** — 5문답
> AI: "양수기동장치 상태는 어떤가요?"
> 💬 *"양수기동장치 정상이에요."*
>
> AI: "라이트커튼 차단 시간 확인됐나요?"
> 💬 *"네, 0.18초 측정했고 정상입니다."*
>
> AI: "금형 교체 시 LOTO는요?"
> 💬 *"개인별 LOTO 락 다 가져왔습니다."*
>
> AI: "보호구 착용 상태는?"
> 💬 *"방진마스크·보안경·안전화 3종 다 착용했습니다."*
>
> AI: "브레이크 정지 시간은?"
> 💬 *"빈 가동 후 정지 시간 측정했고 정상입니다."*

🏗️ **건설 (고소작업)** — 5문답
> 💬 *"안전대 6명 모두 생명줄 체결 1:1 확인했습니다."*
> 💬 *"작업발판 폭 40cm 이상, 안전난간 90cm 설치 완료입니다."*
> 💬 *"개구부 덮개 다 고정했고, 추락방지망 1차/2차 설치했습니다."*
> 💬 *"이동식 비계 캐스터 4개 잠금하고 아웃리거 전개했습니다."*
> 💬 *"공구벨트 스트랩 일체형으로 다 지급했고, 낙하물 방지망 설치돼 있습니다."*

⚓ **중공업 (밀폐공간 작업)** — 5문답 (수치 기반)
> 💬 *"작업허가서 발급했고, 위험요인 3종 다 명기했습니다."*
> 💬 *"4-가스 검지기로 측정했어요. 산소 20.8%, LEL 0%, H2S 1ppm입니다."*
> 💬 *"송풍기 가동 중이고 작업 종료까지 환기 유지합니다."*
> 💬 *"감시인 1명 입구 상주, 무전 5분 간격 호출 시험 통과했습니다."*
> 💬 *"SCBA 1조와 구명로프 입구에 비치돼 있고, 구조자 훈련 끝났습니다."*

🔬 **반도체 (CVD 챔버 PM)** — 5문답 (수치 기반)
> 💬 *"전원·RF·가스 LOTO 다 시행했고, 검전기로 잔류전압 0V 확인했습니다."*
> 💬 *"N2 퍼지 3회 마쳤고, 잔류농도 SiH4 3ppm입니다. 5ppm 미만이라 통과입니다."*
> 💬 *"PFC 농도 측정 후 SCBA 착용하고 진입할 계획이에요."*
> 💬 *"라이너 표면 온도 45°C입니다. 50°C 미만이라 진입 가능합니다."*
> 💬 *"2인 1조로 구성됐고, EHS 1회 입회 예정입니다."*

5/5에 도달하면 화면 상단 단계 스트립이 **"점검"** → **"대응"** 으로 자동으로 점프합니다.

> 💡 AI가 갑자기 "혹시 비상정지 위치는 다 인지하고 계신가요?" 같은 **추가 질문**을 한 번 합니다(누락 위험 점검). 필요하면 답하고, 이미 다뤘으면 *"괜찮습니다, 다음으로"* 라고 하면 AI가 즉시 다음으로 넘어갑니다.

## 8단계 — 사진으로 위험 확인하기 (선택, 권장)

화면 우측 하단 **카메라 아이콘**을 누릅니다.

- **처음이면** 카메라 동의 모달이 뜹니다 → **허용**.
- 작업 영역(설비, 발판, PPE 등)을 촬영합니다.
- 셔터 후 채팅에 **(사진 첨부)** + **"분석 중…"** 메시지가 뜹니다.
- 2~5초 후 AI가 사진 속 위험 요소를 박스(bbox) 와 함께 짚어 줍니다.

> 💡 **TBM 모드 한정** — 신뢰도 70% 이상 위험은 **자동으로 정리본의 "주요 위험요인"에 추가**됩니다. 잘못 잡힌 게 있으면 카드의 **"되돌리기"** 버튼으로 빼면 됩니다. 신뢰도 70% 미만 항목은 카드의 **"+ 체크리스트에 추가"** 버튼으로 수동 추가 가능합니다. **EHS 챗 모드에서는 자동 보강·라벨이 모두 비활성화**되어 카드(사진·bbox·신뢰도 바)만 표시됩니다 — Q&A 모드라 체크리스트가 없는 게 정상입니다.

> 🔊 **AI가 음성으로도 설명합니다** — 사진 분석이 끝나면 AI가 카드 내용을 1~2문장 한국어 음성으로 짚어 줍니다(가장 위험한 항목 우선). TBM 모드는 단계가 **사전 정보·점검** 일 때만 활성화되며, **대응·정리** 단계에선 흐름 보존을 위해 음성 설명이 생략됩니다. EHS는 항상 활성화.

> ⚠️ **반도체는 카메라 아이콘이 기본 비활성화**되어 있습니다. 사이트 정책상 사진 촬영이 금지인 경우가 많기 때문입니다. 사용하려면 **설정 → 카메라 사용 허용 → 반도체 ON** 으로 켜세요.

> 💡 카메라 권한이 거부됐거나 카메라가 없는 PC라면 **"사진 파일 선택"** 폴백이 자동으로 뜹니다. 갤러리 사진으로도 같은 분석이 됩니다.

### 음성으로 위험 추가 묻기
사진 분석 외에 음성으로도 자유롭게 물어볼 수 있습니다.

> 💬 *"이 사진에 있는 케이블 정리는 안전한 편인가요?"*
> 💬 *"화면에 보이는 헬멧 색깔 다른 작업자는 누구신가요?"*

## 9단계 — 정리본 검토

화면 우측 상단의 **"정리본 NN%"** 버튼을 누릅니다. 우측에서 슬라이드 패널이 열리며 8개 항목이 보입니다.

| 항목 | 무엇이 들어가나요 |
|---|---|
| 오늘 작업 내용 | 작업유형 + 특이사항 자동 prefill |
| 평소와 달라진 점 | 신규 자재 + 과거 사고 키워드 자동 prefill |
| 주요 위험요인 | baseline 5건 + 사진 자동 추가 |
| 위험 시나리오 | "~ 시 ~ 발생" 형식 |
| 대응/예방 조치 | "~ 후 ~ 시행" 형식 |
| 보호구/장비 | 안전대·SCBA·LOTO 락 등 |
| 특이사항 | 자유 기재 |
| 참석 확인 | 종료 화면에서 자동 |

음성으로 마무리 발언을 한 번 합니다:
> 💬 *"오늘 마무리할게요. 작업자분들 다 안전하게 진행하시면 좋겠습니다."*

AI가 자동으로 **요약문**(final_summary)을 생성해 정리본 상단에 채워줍니다. 단계 스트립도 **"정리"** 단계로 이동합니다.

## 10단계 — 안전 경고 토스트 (자연 발생)

대화 중 위험한 상황을 묘사하면 AI가 자동으로 **"⚠️ 잠깐만요!"** 토스트를 띄웁니다.

**토스트가 뜨는 멘트 예시**

🏭 제조: 💬 *"라이트커튼이 깜빡거리는데 그냥 우회해서 작업해도 될까요?"*
🏗️ 건설: 💬 *"풍속이 18 m/s로 높아졌는데 작업 계속해도 될까요?"*
⚓ 중공업: 💬 *"감시인이 잠깐 자리 비우는데 그 사이 작업해도 될까요?"*
🔬 반도체: 💬 *"잠깐 동료가 자리 비웠는데 혼자 챔버에 들어가서 작업해도 될까요?"*

> 💡 토스트는 5초 후 자동 사라지거나 ESC/X로 닫을 수 있습니다. 안전 경고는 **채팅 로그에 영구 보존**되어 종료 후에도 기록됩니다.

## 11단계 — TBM 종료 화면

화면 우측 상단 주황색 **"종료"** 버튼을 누릅니다. **"정리"** 화면(`/finish`)으로 이동합니다.

5개 섹션을 차례로 진행합니다.

### §1 요약 편집
AI가 작성한 요약문을 한 번 읽고, 필요하면 한두 문장 추가합니다.
> 💬 (예시 입력) *"프레스 5번 라인 오전 작업 안전 점검 완료. 신규 자재 알루미늄 판재 정상 입고."*

### §2 위험·대응 정리 (8필드)
prefill 된 항목들을 검토하고, 비어 있는 칸이 있으면 **줄바꿈으로 항목 구분**해서 추가합니다.

### §3 참석자 등록 + 서명
이름과 역할을 추가합니다. 각 행 우측의 **○ 아이콘**을 누르면 서명 캔버스가 뜹니다 — 손가락이나 마우스로 서명 후 **확인**.

| 도메인 예시 | 참석자 구성 |
|---|---|
| 제조 | 리더 1 + 작업자 3 = 4명 |
| 건설 | 리더 1 + 작업자 4 + 신호수 1 = 6명 (풍속 강해 신호수 추가) |
| 중공업 | 리더 1 + 작업자 2 + 감시인 1 = 4명 |
| 반도체 | 작업자 2 + EHS 입회 1 = 3명 (2인 1조 + EHS 의무) |

> 💡 두 번째 TBM부터는 **자주 함께 일하는 동료** 칩이 폼 위에 뜹니다. 클릭 한 번으로 빠르게 추가할 수 있습니다.

### §4 미리보기
PDF가 어떻게 나올지 화면에서 미리 확인합니다.

### §5 내보내기 + 종료
- **PDF 내보내기**: `safemate-tbm-<id>.pdf` 다운로드. 이메일이나 프린트로 공유.
- **JSON 내보내기**: 백업·외부 시스템 연동용 데이터 파일.
- **TBM 종료**: 세션을 확정하고 보관함으로 옮깁니다. 홈 화면으로 자동 복귀.

> 📄 PDF에는 한글 폰트(NotoSansKR Regular)가 정상 임베드 됩니다. 폰트가 통째로 들어가므로 **PDF 파일 크기는 약 6 MB** 입니다 — 이메일/메신저 첨부에 무리는 없으나 모바일 데이터 사용량 참고. 만약 PDF에서 한글이 `?`로 깨지거나 글리프가 일부만 보이면, 브라우저 **DevTools(F12) → Console** 을 열어 PDF 재발행 시 `[pdfGenerate]` 로그 확인 — 폰트 fetch 단계, 임베드 단계 어디서 끊겼는지 알려줍니다. 폰트 자산은 `frontend/public/fonts/static/NotoSansKR-Regular.ttf` 위치에 있어야 합니다.

## 12단계 — 과거 기록 보기

홈 화면 → 도구 → **"과거 TBM 기록"** 을 누르면 종료한 TBM 목록이 나옵니다.

- **검색창**: "프레스" / "고소" / "밀폐" / "CVD" 같은 키워드로 필터링.
- **도메인 chip**: "제조" / "건설" 등 도메인별 필터.
- **"보관 포함"** 토글: 활성과 보관 세션 함께 보기.
- 각 행에 도메인 배지 + 작업유형 라벨 + 시간 표시. **EHS 세션도 함께 표시**되며 클릭 시 같은 챗으로 이어 진입.
- 행 우측 **휴지통 아이콘**: 보관함으로 이동(영구 삭제 아님).

영구 삭제는 **설정 → 보관함 영구 삭제** 에서 2단계 confirm을 거쳐야 합니다(prompt에 "삭제" 정확 입력 필요).

> 💡 **홈 화면 "진행 중" 카드는 TBM 세션만 표시**합니다 — EHS 챗은 transient한 Q&A 성격이라 매 진입 시 새 세션을 자동 생성하므로 Resume 카드에 노출되지 않습니다. 이전 EHS 챗을 다시 열고 싶으면 **과거 기록 → "보관 포함"** 토글로 찾아 들어가세요.

## 부록 — EHS 안전 질문하기 (홈 → "EHS 안전 질문하기")

EHS는 체크리스트 없이 안전 지식 Q&A만 하는 모드입니다. 사용 흐름:

1. 홈 → **"EHS 안전 질문하기"** → 자동으로 새 EHS 세션 생성 + 챗 화면(`/ehs/<id>`) 진입.
2. 마이크 켜고 자유롭게 질문:
   > 💬 *"고소작업 안전대 차종별 적용 기준 알려주세요."*
3. **추천 질문 칩**(채팅 아래 "묻기 좋은 질문 N개" 영역)을 한 번 누르면 즉시 AI가 음성+텍스트로 답변합니다 — 마이크가 OFF여도 동작. 답변 중 다른 칩을 누르면 진행 중인 답변이 중단되고 새 질문으로 전환됩니다.
4. AI가 답변하면서 관련 법규·문서를 화면에 카드로 함께 표시. 음성으로도 *"관련 자료를 화면에 함께 표시했습니다"* 라고 짧게 짚어 줍니다.
5. 사진을 첨부하면 AI가 위험을 검출 → 카드(사진·bbox·hazard 카드)와 함께 음성으로 1~2문장 설명. **단, 체크리스트가 없는 모드라 "+ 체크리스트에 추가"·"되돌리기" 같은 버튼은 노출되지 않습니다** — 카드와 음성 답변만 받아보세요.
6. 종료할 때는 **상단 우측 "종료" 버튼** 누르면 confirm 후 홈 화면으로 돌아갑니다(TBM의 정리/서명/PDF 흐름은 EHS에 없음 — Q&A 모드라 의도된 동작). 좌측 상단 홈 아이콘으로도 동일하게 종료 가능. 어느 쪽이든 IndexedDB에 자동 보존되어 **과거 기록**에 남습니다.

## 문제 해결 (FAQ)

**Q. 음성이 인식 안 돼요.**
→ 마이크 버튼이 주황색인지 확인. 회색이면 한 번 더 누르세요. 그래도 안 되면 브라우저 주소창 자물쇠 → 마이크 권한 확인.

**Q. AI가 한국어인데 영어로 답해요.**
→ 화면 상단 언어 칩이 "한국어"로 되어 있는지 확인. "English"로 되어 있으면 클릭해서 한국어로 변경.

**Q. 사진 찍어도 분석이 안 돼요.**
→ 반도체라면 설정 → 카메라 사용 허용 → 반도체 ON 확인. 다른 도메인이면 OPENAI 키 설정 필요(관리자에게 문의).

**Q. 위험 추천이 안 떠요.**
→ 작업 카드를 다시 한 번 누르거나, 위험 추천 섹션의 **🔄 다시 받기** 클릭.

**Q. 진행 중이던 TBM이 사라졌어요.**
→ 홈 화면 상단의 **"진행 중"** 카드를 누르세요. 자동 저장되므로 새로고침해도 사라지지 않습니다. 만약 정말 사라졌다면 **과거 TBM 기록**에서 검색.

**Q. 종료 후 PDF를 다시 받고 싶어요.**
→ 과거 TBM 기록에서 해당 세션을 누르면 다시 종료 화면으로 들어가 재발행 가능합니다.
